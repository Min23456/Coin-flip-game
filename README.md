# 🪙 Coin Flip Game
A simple and fun coin flip game built for quick interactions and learning purposes. Flip the coin and see if you get **Heads** or **Tails**!

---

## 🎯 Project Overview
This project serves as a practical example for beginners to understand:
- **DOM Manipulation**: How to change UI elements based on logic.
- **Math.random()**: Implementing simple probability in JavaScript.
- **CSS Animations**: Creating a realistic "flipping" effect.

## 🎮 How to Play
1. Click the **Flip Coin** button.
2. The coin will rotate rapidly using CSS keyframes.
3. Once the animation stops, the result (Heads or Tails) will be displayed.
4. Keep track of your winning streak!
