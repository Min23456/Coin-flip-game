# 🪙 Coin Flip Game

A simple and fun coin flip game built for quick interactions and learning purposes. Flip the coin and see if you get **Heads** or **Tails**!

---

## 📌 Features

- Random coin flip outcome (Heads or Tails)
- Simple and lightweight
- Beginner-friendly project
- Easy to customize and expand
